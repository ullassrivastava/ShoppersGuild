
# Shoppers_Guild
## Introduction
Shoppers Guild is an eCommerce web application made with Django-based web application that allows users to browse through the website and purchase different
items enlisted on the site. The site allows the user to Log in and Sign Up, so that the vendor has potential to keep track of the users.

## Technologies Used:
Python, CSS, HTML, JavaScript, Bootstrap

## Features
1. Run the server
2. Browse the website
3. Navigate carousel
4. Play with the tabs
5. Expand the lists
6. Buy the enlisted product
